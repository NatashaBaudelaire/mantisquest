# MANTIS QUEST - Defect Tracker Simulator

Mantis Quest is a retro arcade-inspired educational game designed to teach software defect tracking concepts through gameplay mechanics, quizzes, simulations, and boss battles.

The project transforms QA and bug management into an interactive arcade experience.

---

# Features

- Retro arcade interface
- Quiz system about QA and MantisBT
- XP and score progression
- Hall of Fame ranking
- Bug tracking simulation
- Boss Battle against critical bugs
- CRT/pixel visual effects
- Retro sound effects using Web Audio API

---

# Technologies Used

- HTML5
- CSS3
- JavaScript Vanilla
- Web Audio API
- Tabler Icons

# Educational Purpose

The project was created to teach concepts such as:

- Quality Assurance (QA)
- Bug Tracking
- Severity Levels
- Software Defect Workflow
- MantisBT usage
- Defect lifecycle

through gamification.

---

# Game Modes

## Quiz Mode
Answer questions about:
- MantisBT
- QA
- Defect management
- Severity classification

## Demo Mode
Simulate a real bug tracking environment:
- Create bugs
- Define severity
- Resolve issues
- Filter tickets

## Boss Battle
Fight against "The Big Bug" using:
- Register defect
- Define severity
- Assign developer
- Resolve and close

## Memorial
Technical overview about:
- Strengths
- Weaknesses
- Workflow
- Use cases of MantisBT

---

# Scoring System

Players earn points by:
- Correct answers
- Managing bugs
- Defeating bosses
- Exploring the system

Ranks include:
- Junior Development
- QA Hero
- Bug Master
